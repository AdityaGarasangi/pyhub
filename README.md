## PyHub
A collection of Python projects I’ve built along the way — usually because writing something in Python felt like the easiest way to make something a bit simpler for myself.
Some started as quick ideas, some turned into bigger projects.
Just a place where the things I build using Python end up living.

### Projects

- LogAnalyzerCLI - Built this after getting tired of digging through large log files trying to find errors or patterns manually. This parses logs and gives quick summaries so it's easier to see what's going on.

- ContainerWatchdog - I kept finding out my containers had crashed hours ago and i had no idea. so i built this python project over a weekend and honestly it's one of those tools i use every single day without thinking about it.
